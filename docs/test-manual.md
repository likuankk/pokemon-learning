## 常见问题排查

| 现象 | 可能原因 | 解决方法 |
|------|---------|---------|
| 登录提示"用户不存在" | 执行了 `--clean` 或场景切换失败 | 重新执行场景脚本 |
| 战斗区域全锁定 | total_wins不够解锁 | 使用 `battle-mid` 或 `battle-endgame` 场景 |
| 进化按钮灰色 | 等级或碎片不足 | 使用 `evolution` 场景 |
| 审核中心无任务 | 没有submitted状态任务 | 使用 `tasks` 场景或孩子端先提交任务 |
| 小屋为空 | 没有购买装饰品 | 使用 `house` 或 `full` 场景 |
| 技能只有1-2个 | battle_level不够解锁更多技能槽 | 使用 `battle-endgame`(全4技能) |
