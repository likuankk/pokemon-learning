## 🐛 发现的 Bug

### Bug 1: "已使用undefined分钟" 提示 (P1 - 中等)
- **现象**: 页面底部频繁出现黄色提示"🤯 已使用undefined分钟，注意休息哦~"
- **出现位置**: 任务页、进化页、喂食页、战斗页等几乎所有页面
- **原因**: 使用时长变量未正确传值，显示为 `undefined`
- **影响**: 用户体验差，提示文案不正确

### Bug 2: PP 值不更新 (P2 - 低)
- **现象**: 战斗中使用技能后，PP (使用次数) 显示未减少，仍为满值
- **出现位置**: 战斗界面技能按钮
- **可能原因**: PP 消耗后UI未重新渲染

### Bug 3: 成就总数与手册不一致 (P3 - 待确认)
- **现象**: 手册描述成就总数为58个，实际显示36个
- **出现位置**: 图鉴成就页面
- **可能原因**: 成就定义数据可能更新了，手册需同步
